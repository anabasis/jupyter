한글 형태소 분석기 성능¶

  • 띄어쓰기 성능 : Kkma = Komoran > Okt > Hannanum
  • 형태소 분석 성능 :
  • (문어체) Kkma = Komoran > Hannanum > Okt
  • (구어체) Komoran > Okt = Hannanum > Kkma

tokens1 = list() # 원시 어절
tokens2 = list() # 품사분석 (튜플)
tokens3 = list() # 형태소
tokens4 = list() # 품사

for sentence in sent_tokenize(corpus):
    tokens = word_tokenize(sentence)
    for token in tokens:
        tokens1.append(token)
        tokens2.extend(komoran.pos(token))
        tokens3.extend([_[0] for _ in komoran.pos(token)])
        tokens4.extend([_[1] for _ in komoran.pos(token)])
